Christianity Crisis Center - Color Accessibility Test

Light Mode Color Combinations

Primary Red (#E31E24) on White (#FFFFFF)

This is the primary text color for headings and important content. It should have high contrast for readability.

Used for: Navigation links, headings, call-to-action buttons
Dark Gray (#2B2B2B) on White (#FFFFFF)

This is the main body text color. It provides excellent readability while being easy on the eyes for long reading sessions.

Used for: Body text, paragraphs, descriptions
Primary Red (#E31E24) on Gold (#D4AF37)

This combination is used in the navigation bar. The red text should be clearly visible against the gold background.

Used for: Navbar text, gold section content
White (#FFFFFF) on Primary Red (#E31E24)

This high-contrast combination is perfect for buttons and call-to-action elements that need to stand out.

Used for: Primary buttons, alert messages
Light Gold (#FFD700) on Primary Red (#E31E24)

This combination provides accent highlights while maintaining the brand colors.

Used for: Accent text, special highlights
Primary Gold (#D4AF37) on White (#FFFFFF)

Gold text on white background for decorative elements and secondary content.

Used for: Accent elements, secondary buttons

Dark Mode Color Combinations

Primary Red (#E31E24) on Dark Background (#1A1A1A)

Primary red maintains visibility in dark mode while preserving brand identity.

Used for: Dark mode navigation, headings
Light Gray (#F5F5F5) on Dark Background (#1A1A1A)

Light text on dark background provides comfortable reading experience in dark mode.

Used for: Dark mode body text

Accessibility Guidelines

WCAG 2.1 Contrast Requirements

  • AA Level (Minimum): 4.5:1 for normal text, 3:1 for large text
  • AAA Level (Enhanced): 7:1 for normal text, 4.5:1 for large text

Color Combination Analysis

  • Primary Red on White: High contrast - excellent readability
  • Dark Gray on White: Optimal contrast for body text
  • White on Primary Red: Maximum contrast for CTAs
  • Primary Red on Gold: Good contrast - may need testing

Recommendations

  • Use Primary Red (#E31E24) for important headings and navigation
  • Use Dark Gray (#2B2B2B) for body text and descriptions
  • Ensure buttons have sufficient padding for touch targets
  • Test with color blindness simulation tools
  • Provide focus indicators for keyboard navigation
